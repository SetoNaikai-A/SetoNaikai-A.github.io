    Markdown的基本使用

    发表于 2021-07-16 更新于 2021-07-19 分类于 前端

    1.Markdown是什么

    Markdown是一种轻量级的标记语言，通过Markdown语法编辑文本，能够使文本呈现清晰的结构

    2.Mark的基本语法

    2.1标题

    Markdown中设置标题需要通过在文本前添加# #号的数量代表标题的等级 Markdown支持1~6级标题

    效果为

    1级标题

    2级标题

    3级标题

    4级标题

    5级标题
    6级标题

    2.2字体

    需要加粗的字体前后需要用2个*号包起来
    *需要加粗的字体*
    需要加粗的字体

    需要使用斜体的字体前后需要用1个*号包起来
    *需要使用斜体的字体
    需要使用斜体的字体

    需要使用删除线的字体前后需要用2个~号包起来
    ~需要使用删除线的字体~
    需要使用删除线的字体

    PS：
    Markdown中的许多符号都有特殊的含义 要正确地输入这些字符就需要使用转义字符
    在需要输入的字符前添加\

    2.3分割线

    在需要添加分割线处加入3个*或3个-

    2.4代码块

    单行代码前后需要用1个`号包起来
    hexo s -p 5555

    多行代码前后需要用3个`号包起来 且3个`应该独占1行

    2.5引用

    在需要引用的文本前加> >的数量代表引用的级别
    >需要引用的文本

    需要引用的文本

    2.6图片

    ![alt text 属性文本](图片地址”optional title 可选标题”)
    alt text 属性文本
    为描述图片的关键词 当图片无法显示将会显示alt属性文本

    optional title 可选标题
    当鼠标悬停在图片上时将会显示optional title
    ![水面向こう](https://i.loli.net/2021/07/19/AEqZtxHhXDLlKVR.jpg)

    PS：
    Markdown中插入的图片的地址可以是本地路径或服务器的网络地址
    博客中的图片往往会存放在一种专门存储图片的服务器中 即图床
